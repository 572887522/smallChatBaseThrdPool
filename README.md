# smallChatBaseThrdPool

Linux平台，基于线程池和socket的简单的网络通信实现，供以学习参考。





# 如何使用

### 编译

```makefile
make
```

### 开一个窗口启动服务器端

```shell
./server
```

### 开两个窗口启动客户端

```shell
./client
```
